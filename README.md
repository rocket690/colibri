# Клиника Колибри — статический сайт

Файлы этого проекта готовы для загрузки в репозиторий GitHub и публикации через GitHub Pages.

Структура:

```
/ (root)
  index.html
  about.html
  services.html
  contact.html
  privacy.html
  css/style.css
  js/main.js
  assets/img/*
```

Инструкция по публикации:
1. Создайте репозиторий на GitHub (public).
2. Загрузите файлы (через `git` или веб-интерфейс).
3. В Settings → Pages выберите ветку `main` и папку `/ (root)` → Save.
4. Через минуту сайт будет доступен по адресу `https://<ваш-ник>.github.io/<repo>/`.

Если нужно — могу адаптировать дизайн, поменять цвета, добавить логотип или интегрировать контактную форму через Formspree.
